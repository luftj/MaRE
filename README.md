# MaRE

Extract water bodies from topographic maps and match them to OSM data for georeferncing.

WIP.

---

## Installation

Requires
* Python3
* python-opencv

```$ python3 -m pip install -r requirements.txt ```


## Usage

`$ python3 main.py [image path] [colour balance percent]`
